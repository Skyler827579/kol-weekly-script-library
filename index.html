自动化流程：每位 KOL 每周 3 篇基础稿件，另加周一、周三、周五行情解读。

基础稿件是固定教学框架；行情解读会结合每个人风格讲近两天或近一周行情。页面会读取实时行情数据，把 BTC / ETH / SOL / Gold 的价格、涨跌幅、RSI、MACD、ATR 写进口播稿。

How to use

使用方式

  1. 先选择 William、KC 或 Caven，进入对应的脚本库。
  2. 先看“本周三篇基础稿件”，再看“周一 / 周三 / 周五近期行情解读”。
  3. 录制前点击“刷新行情”，页面会重新读取价格、RSI、MACD、ATR，并同步更新脚本里的行情插入和英文口播稿。
